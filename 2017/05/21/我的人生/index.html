我的人生

有一个令人吃惊的事实：人生只有900个月，这对于大部分人来说是一样的，在一张A4纸上画个30X30的表格， 过去一个月就打个勾，你会发现，生命过得太快了。之前有想过针对这个概念设计一个手机应用，输入你的 出生日期，系统会自动计算你的生命已走过了多少个月，还有多少月是需要好好珍惜的，每天可以通过应用中 的笔记、录音、照相功能来记录生活的点点滴滴。应用的背景画面是一个人站在茫茫的天地间，远处是一些枯树。 一句话提示人们：Your time is limited, so don`t waste it living someone else`s life.　

------------摘自知乎 《人生短短900月》
